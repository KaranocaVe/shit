apply.ignoreWhitespace::
	When set to 'change', tells 'shit apply' to ignore changes in
	whitespace, in the same way as the `--ignore-space-change`
	option.
	When set to one of: no, none, never, false, it tells 'shit apply' to
	respect all whitespace differences.
	See linkshit:shit-apply[1].

apply.whitespace::
	Tells 'shit apply' how to handle whitespace, in the same way
	as the `--whitespace` option. See linkshit:shit-apply[1].
