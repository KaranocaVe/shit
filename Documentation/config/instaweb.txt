instaweb.browser::
	Specify the program that will be used to browse your working
	repository in shitweb. See linkshit:shit-instaweb[1].

instaweb.httpd::
	The HTTP daemon command-line to start shitweb on your working
	repository. See linkshit:shit-instaweb[1].

instaweb.local::
	If true the web server started by linkshit:shit-instaweb[1] will
	be bound to the local IP (127.0.0.1).

instaweb.modulePath::
	The default module path for linkshit:shit-instaweb[1] to use
	instead of /usr/lib/apache2/modules.  Only used if httpd
	is Apache.

instaweb.port::
	The port number to bind the shitweb httpd to. See
	linkshit:shit-instaweb[1].
