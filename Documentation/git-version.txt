shit-version(1)
==============

NAME
----
shit-version - Display version information about shit

SYNOPSIS
--------
[verse]
'shit version' [--build-options]

DESCRIPTION
-----------
With no options given, the version of 'shit' is printed on the standard output.

Note that `shit --version` is identical to `shit version` because the
former is internally converted into the latter.

OPTIONS
-------
--build-options::
	Include additional information about how shit was built for diagnostic
	purposes.

shit
