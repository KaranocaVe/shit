shit-merge-one-file(1)
=====================

NAME
----
shit-merge-one-file - The standard helper program to use with shit-merge-index


SYNOPSIS
--------
[verse]
'shit merge-one-file'

DESCRIPTION
-----------
This is the standard helper program to use with 'shit merge-index'
to resolve a merge after the trivial merge done with 'shit read-tree -m'.
