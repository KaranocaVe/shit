shit-citool(1)
=============

NAME
----
shit-citool - Graphical alternative to shit-commit

SYNOPSIS
--------
[verse]
'shit citool'

DESCRIPTION
-----------
A Tcl/Tk based graphical interface to review modified files, stage
them into the index, enter a commit message and record the new
commit onto the current branch.  This interface is an alternative
to the less interactive 'shit commit' program.

'shit citool' is actually a standard alias for `shit gui citool`.
See linkshit:shit-gui[1] for more details.

shit

Part of the linkshit:shit[1] suite
